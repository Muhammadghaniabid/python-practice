# 🎓 School Management Mini System (Python)

## 📌 About Project

This is a simple Python project where users can act as:

* Admin
* Teacher
* Student

Each user can:

* 👀 View their list
* ➕ Add new items to the list

This project is made for learning basic Python concepts.

---

## 🚀 Features

* Uses **while loop** for continuous running
* Uses **match-case** (Python pattern matching)
* Uses **lists**
* Uses **append()** to add new data
* Simple user interaction with input

---

## 🧠 Concepts Used

* Lists in Python
* Loops (`while`)
* Conditional logic using `match-case`
* User input handling

---

## ▶️ How to Run

1. Install Python (3.10 or above)
2. Download this file
3. Run in terminal:

```
python school_system.py
```

---

## 💻 Example

```
Enter your type: admin

['Fee', 'Attendance', 'School']

Enter your type: admin

Enter your type: admin
```
